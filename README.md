# Análisis de Datos - Alura Store 📊

Este proyecto analiza el rendimiento de 4 tiendas para decidir cuál debe venderse para iniciar un nuevo negocio.

## Herramientas
* Python (Pandas y Matplotlib)
* Google Colab

## Resumen de Resultados
* **Tienda con menores ingresos:** Tienda 4 ($1,038,376,400)
* **Mejor calificación:** Tienda 3 (4.05)
* **Recomendación:** Vender la **Tienda 4** para minimizar el impacto financiero.
